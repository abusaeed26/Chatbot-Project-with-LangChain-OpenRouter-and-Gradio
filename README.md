# 🤖 AI Chatbot with LangChain, OpenRouter, and Streamlit

This project is a conversational AI chatbot built using LangChain, integrated with OpenRouter (or Hugging Face models), and deployed using a Streamlit web interface. The chatbot can interact with users and maintain conversation history for better responses.

---

## 🚀 Features

- 💬 Interactive chatbot interface
- 🧠 Powered by LangChain (ConversationChain + Memory)
- 🔗 OpenRouter / OpenAI / HuggingFace integration
- 🗂️ Conversation memory support
- ⚡ Fast and simple UI using Streamlit

---

## 🛠️ Tech Stack

- Python
- Streamlit
- LangChain
- OpenRouter / OpenAI / HuggingFace
- dotenv
